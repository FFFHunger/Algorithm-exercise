计算系数From 西部314背景 Background
NOIP2011 day2 第一题
描述 Description
给定一个多项式(ax+by)^k,请求出多项式展开后x^n*y^m项的系数。输入格式 InputFormat
共一行，包含5 个整数，分别为 a ，b ，k ，n ，m，每两个整数之间用一个空格隔开。 输出格式 OutputFormat
输出共1 行，包含一个整数，表示所求的系数，这个系数可能很大，输出对10007 取模后的结果。 


样例输入 SampleInput [复制数据]
1 1 3 1 2 

样例输出 SampleOutput [复制数据]
3

数据范围和注释 Hint
对于30% 的数据，有 0 ≤k ≤10 ； 
对于50% 的数据，有 a = 1，b = 1； 
对于100%的数据，有 0 ≤k ≤1,000，0≤n, m ≤k ，且n + m = k ，0 ≤a ，b ≤1,000,000。
